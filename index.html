Privacy Policy for "Transcript to Article" Chrome Extension

Last Updated: March 2025

Introduction

Thank you for using the "Transcript to Article" Chrome extension ("the Extension"). This privacy policy explains how we handle user data when you use the Extension. Our goal is to be transparent about the data we collect, how we use it, and how we protect your privacy.

1. Data We Collect

The Extension collects the following user data:

  • OpenAI API Key: When you input your OpenAI API key in the Extension’s popup, it is stored locally in your browser using Chrome’s chrome.storage API. This key is used to make API calls to OpenAI for generating articles from YouTube transcripts.

2. How We Use Your Data

The OpenAI API key is used solely to authenticate and make requests to OpenAI’s API for the purpose of generating SEO-optimized blog articles from YouTube transcripts. The key is sent directly to OpenAI during API calls and is not stored or transmitted elsewhere by the Extension.

3. Data Storage and Security

  • The OpenAI API key is stored locally on your device using chrome.storage.sync (or chrome.storage.local if sync is disabled). This means the key may sync across your devices if Chrome Sync is enabled.
  • We do not store, log, or transmit your API key to any third parties other than OpenAI for the purpose of API requests.
  • The Extension does not collect or store any other personal data, such as your browsing history, YouTube video content, or generated articles, beyond what is temporarily processed during operation.

4. Third-Party Services

  • OpenAI: The Extension uses OpenAI’s API to generate articles. When you provide an OpenAI API key, it is sent to OpenAI to authenticate API requests. Any transcript data extracted from YouTube is sent to OpenAI for processing. Please review OpenAI’s Privacy Policy to understand how they handle your data.
  • YouTube: The Extension extracts transcripts from YouTube video pages (e.g., https://www.youtube.com/watch?v=...) if captions are enabled. This process occurs locally in your browser and does not involve storing or transmitting the transcript beyond the API call to OpenAI.

5. Data Sharing

We do not share your data with any third parties except as described above (OpenAI for API requests). The Extension does not collect or share personal information such as your name, email, or browsing history.

6. User Control

  • You can view, update, or remove your OpenAI API key at any time through the Extension’s popup interface.
  • To remove the stored API key, you can clear the Extension’s storage via Chrome’s settings (chrome://extensions/ > "Transcript to Article" > "Clear storage").

7. Cookies

The Extension does not use cookies.

8. Changes to This Privacy Policy

We may update this privacy policy from time to time to reflect changes in the Extension’s functionality or legal requirements. Any updates will be posted at this URL, and the "Last Updated" date will be revised accordingly.
